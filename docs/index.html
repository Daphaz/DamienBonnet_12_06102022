SportSee - Front-end dashboard

This repo contains all the source code to run the front-end app for the sports analytics dashboard SportSee.

Prerequisites
  • Nodejs (version 14.17.3)
  • npm (version 6.14.13)

We strongly recommend using a Node version manager like nvm to install Node.js and npm. After nvm installation, in this root folder launch :

nvm use

Launch the project

Don't forget to install and laung the sportSee api

  • clone this repo
  • The npm i command install the dependencies.
  • The npm start command will allow you to run thi application.
